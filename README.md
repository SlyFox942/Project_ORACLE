# 🔮 Project ORACLE

A Python analytics platform for exploring historical lottery data.

## Features

- 📊 Interactive Dashboard
- 🎱 Mega Millions Analysis
- 🔴 Powerball Analysis
- 🔥 Hot & Cold Numbers
- 📈 Statistics & Charts
- 🔮 ORACLE Score
- 🎲 Experimental Pick Generator
- 📥 CSV Data Import
- 🗄 SQLite Database
- 📊 Plotly Visualizations

## Built With

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
- NumPy
- Scikit-Learn

Project ORACLE is for educational, statistical analysis, and entertainment purposes only. Lottery drawings are designed to be random. This application does not predict future winning numbers or improve the odds of winning.
